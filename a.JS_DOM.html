Javascript를 이용한 HTML태그 접근

자바스크립트를 이용하여 HTML태그를 동적으로 처리하기 위해서는 메소드를 통하여 HTML에 접근해야함.

메소드

1.getElementById("아이디명") : 태그의 id속성값을 이용해서 태그 요소 객체 정보를 가져옴
2.getElementsByName("이름명") : 태그의 name속성값을 시용해서 태그 요소 객체 정보를 배열에 담아 가져옴(여러개 존재할 수 있음)
3.getElementsByTagName("태그명") : 태그명을 이용해서 태그들의 객체 정보를 배열에 담아 가져옴(여러개 존재할 수 있음)
